# 🌊 VocaVault: 100% Vibe Coded

> "코드는 수단일 뿐, 중요한 건 바이브(Vibe)다."

**VocaVault**는 전통적인 개발 방식에서 벗어나, 인간의 의도(Intention)와 AI의 실행력(Execution)이 만나 탄생한 **100% 바이브 코딩** 프로젝트입니다. 보일러플레이트를 짜는 고통 대신, "이런 느낌의 단어장이 갖고 싶어"라는 영감에서 시작해 단 몇 번의 대화로 완성되었습니다.

---

## ✨ Philosophy of Vibe Coding

이 프로젝트는 다음과 같은 원칙으로 빌드되었습니다:

1. **Zero Boilerplate**: 한 줄의 설정도 손으로 직접 쓰지 않았습니다. 모든 구조는 대화를 통해 생성되었습니다.
2. **Iterative Vibe Check**: 기능을 만들고, 써보고, "음.. 이건 좀 더 로그(Log)적인 성장이 필요해"라는 직관적인 피드백으로 시스템을 정교화했습니다.
3. **Prompt-Native**: 코드를 읽는 시간보다 의도를 전달하는 시간에 집중했습니다.

---

## 🚀 Core Features

- **📂 Topic-Based Vault**: 단어를 주제별로 묶어 관리합니다. (음식, 여행, 비즈니스 등)
- **🧠 3-Way Quiz Mode**:
  - *Flashcards*: 전통적인 카드 뒤집기 방식.
  - *Word Quiz*: 뜻을 보고 단어를 맞히는 4지선다.
  - *Definition Quiz*: 단어를 보고 뜻을 맞히는 4지선다.
- **🎮 XP & Level System**: 학습할 때마다 경험치를 얻고 레벨업하세요.
- **📈 Logarithmic Growth Curve**: 레벨이 오를수록 필요 경험치가 늘어나지만, 그 상승폭은 점차 완만해지는 "인간적인" 성장 곡선을 적용했습니다. 
  - 공식: `R(L) = 500 + 250 * ln(L)`

## 🛠 Tech Stack (The Vibe Engine)

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Database**: [Prisma](https://www.prisma.io/) with SQLite (Zero-config local DB)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Collaborator**: Gemini CLI

## 🛠 Getting Started

바이브를 직접 느껴보려면:

1. **의존성 설치**:
   ```bash
   npm install
   ```

2. **데이터베이스 초기화 (Vibe Sync)**:
   ```bash
   npx prisma db push
   ```

3. **개발 서버 실행**:
   ```bash
   npm run dev
   ```

4. **접속**: `http://localhost:3000` 에서 당신의 단어 보관소를 확인하세요.
